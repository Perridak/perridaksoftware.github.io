Get Help

If you need assistance with any Perridak Software product, we recommend starting with the resources already available on this website. Most questions can be answered quickly through the sections below.

Frequently Asked Questions (FAQ)

Our FAQ page covers common topics such as installation, licensing, troubleshooting steps, and general product behaviour. It’s often the fastest way to find answers to routine questions.

Documentation and Manuals

The Documentation section provides access to software manuals, technical guides, and API references. These resources are updated as new versions of our applications are released and are designed to help you understand features, configuration options, and integration points.

Reporting a Problem

For issues specific to an individual application, problem reports will be handled through GitHub Issues once each project is publicly available. This allows you to track progress, provide additional details, and see known issues or planned fixes. Each application’s page will link directly to its issue tracker when live.

Contacting Support

If you cannot find the information you need in the FAQ or documentation, or if your question relates to something not covered by GitHub Issues, you can contact us directly.

Support email:

You may also use the Contact Us page, which includes a simple web form for general enquiries or private communication.

Before You Contact Us

To help us respond efficiently, please check the following first:

  • Review the FAQ for common questions
  • Consult the documentation for detailed instructions
  • For app‑specific bugs, check whether an issue already exists on GitHub

If you still need help, feel free to reach out. We aim to provide clear, practical support while keeping your privacy protected and your data minimised.
