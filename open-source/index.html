Open Source Acknowledgements

Perridak Software could not have started without the support of exceptional open source tools created and maintained by dedicated developers and communities around the world. We do not claim to be endorsed by the owners of the software below, but we are grateful for their work and proudly acknowledge the technologies that helps make our products and business possible.

CmapTools

A concept‑mapping tool used for visualising ideas, workflows, and knowledge structures. It supports structured thinking and collaborative planning.

Website: https://cmap.ihmc.us/

Dart

A modern, expressive programming language optimised for UI development. It powers cross‑platform apps built with Flutter and other toolchains.

Website: https://dart.dev/

Eleventy (11ty)

A fast, flexible static site generator that powers this website. Its simplicity and extensibility make it ideal for clean, maintainable content‑driven sites.

Website: https://www.11ty.dev/

MKDocs

A documentation‑focused static site generator written in Python. It provides a clean, efficient workflow for producing structured technical documentation.

Website: https://www.mkdocs.org/

Pennivo

A lightweight tool for structured writing and content organisation. Useful for drafting, outlining, and maintaining consistent documentation.

Website: https://pennivo.com/

PlantUML

A text‑based diagramming tool that generates UML and other diagrams from simple, readable syntax. Ideal for documentation and architecture design.

Website: https://plantuml.com/

Python

A versatile programming language used across automation, tooling, data processing, and backend workflows throughout the software industry.

Website: https://www.python.org/

Redmine

A powerful open source project management and issue tracking platform. It supports multi‑project workflows, wikis, and flexible issue tracking.

Website: https://www.redmine.org/

Visual Studio Code

Visual Studio Code is a fast, lightweight, cross‑platform code editor with powerful built‑in tools for debugging, source control, extensions, and language support, and endless customisation possibilties via extensions.

Website: https://code.visualstudio.com
